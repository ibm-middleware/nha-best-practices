# IBM MQ NativeHA Reference Automation

A pure GitOps-aligned Reference Implementation to provision, govern, and lifecycle IBM MQ NativeHA Queue Managers on Kubernetes/OpenShift distributions using Kustomize and ArgoCD.

## Motivation & Architecture

This repository offers an infrastructure-as-code pattern utilizing Ansible to programmatically generate pure-YAML definitions for MQ components without embedding cluster-specific logic, meaning a downstream CD system (like ArgoCD or Flux) can seamlessly apply logic against clusters without manual interaction.

The implementation relies on:
* **Cert-Manager / External Secrets Operator**: Automatically injecting cryptographic material into the queue managers.
* **ArgoCD**: Performing continuous reconciliation against the configurations generated by the playbooks.
* **Ansible**: Bootstrapping instances, computing correct storage allocations dynamically based on size.
